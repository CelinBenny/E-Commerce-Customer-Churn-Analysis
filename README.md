# E-Commerce-Customer-Churn-Analysis

## Project Overview : 
This project focuses on analyzing customer churn behavior in an e-commerce business using SQL. The objective is to identify patterns, trends, and factors influencing customer attrition by performing data cleaning, transformation, exploration, and analysis on historical customer transaction data.
The analysis helps businesses understand customer behavior and supports the development of targeted retention strategies to reduce churn and improve long-term profitability.

## Problem Statement :
Customer churn is one of the major challenges faced by e-commerce companies. This project investigates customer attrition patterns using customer demographics, satisfaction scores, payment preferences, order behavior, and engagement metrics.
The goal is to:
- Understand factors contributing to customer churn
- Analyze customer purchasing and engagement behavior
- Generate actionable insights for customer retention
- Support business decision-making through SQL-based data analysis

## Project Workflow :
- Data Cleaning:
    - Handling Missing Values
    - Outlier Treatment
    - Handling Inconsistencies
- Data Transformation:
    - Column Renaming
    - Creating New Columns
    - Column Dropping
- Data Exploration and Analysis:
  Performed SQL analysis to answer business questions such as:
     - Count of churned vs active customers
     - Average tenure of churned customers
     - Percentage of churned customers who complained
     - Most preferred payment modes
     - Customer behavior based on city tiers
     - Coupon usage trends
     - Satisfaction score analysis
     - Cashback behavior analysis
     - Order category performance
     - Distance-based churn analysis
     - Customer returns analysis using JOIN operations

## Key Insights :

- Identified customer segments with higher churn probability
- Found preferred payment methods among active customers
- Analyzed customer satisfaction and complaint behavior
- Discovered churn trends across different city tiers
- Evaluated cashback and coupon usage patterns
- Categorized customers based on warehouse distance impact
